# Course-Match
A great app to track courses your friends are taking, to be friends with people who are taking the same course as you.

## A great place to be friend with people who are pursuing the same course as you

<hr/>

> ### How to set up
- Fork the repository
- clone the repository
- open a terminal navigate to the project directory
- type `yarn install` 
- on the root of the project type `yarn dev`
